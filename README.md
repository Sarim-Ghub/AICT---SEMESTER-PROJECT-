**Shajar** is a sustainability-focused static website developed as part of the **AICT Semester Project**. The project aims to promote environmental awareness by highlighting tree plantation initiatives, community participation, and eco-friendly practices. The website is designed with a clean layout, clear navigation, and structured content to provide an engaging user experience.

## Project Goals & Objectives

The main objectives of this project are:

- To apply fundamental **web development concepts**
- To design a **multi-page static website**
- To implement **structured layouts** using HTML
- To style pages using **CSS**
- To integrate **media content** such as images and videos
- To host and manage the project using **GitHub Pages**

## Key Features

- **Responsive Navigation Bar**  
  Easy navigation across Home, Services, Gallery, About Us, Contact, and Table pages.

- **Hero Section**  
  Introduces the project theme and mission with visually appealing content.

- **Services Page**  
  Highlights plantation services, environmental initiatives, and sustainability efforts.

- **Gallery Page**  
  Displays images and videos showcasing project activities and events.

- **About Us Page**  
  Explains the purpose, mission, and vision of the Shajar initiative.

- **Table Page**  
  Presents structured data related to environmental topics.

- **Footer Section**  
  Contains quick navigation links and legal information.
